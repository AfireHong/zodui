# 基本方向

目标是基于 zod 实现基于类型的 UI 映射，并且在此基础上提供丰富的各个层面的扩展性 API 以供用户使用，封装。

## 基本思想

对于 UI 来说，分为俩种类型的元素

* Viewer：展示型元素，如文本、图片、视频等
* Controller：控制型元素，如按钮、输入框等

从组件设计角度分析，上面俩种类型的元素的本质性区别为对数据的可控性，
即 Viewer 只能展示数据，而 Controller 可以控制数据。
